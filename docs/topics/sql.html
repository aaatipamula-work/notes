Database

We are using MSSQL or Microsoft SQL Server.

MSSQL uses a superset of SQL called T-SQL or Transact SQL.

Being a superset of SQL it functions similarly at its core and anything you've already learned about SQL should apply. The only statement I've found that does not work is the LIKE statement.

I suggest downloading SSMS or SQL Server Management Studio to interact with our databases.
